## 🎬 How It Works

```
User clicks level button
        ↓
Popup slides up with animation
        ↓
Shows: Level Name, Description, Score, ⭐⭐⭐
        ↓
User clicks:
├─ "Proceed" → Level loads
└─ "Close" → Back to menu
```
