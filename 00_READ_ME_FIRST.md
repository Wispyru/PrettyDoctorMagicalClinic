## ⭐ Star System Explained

```
Score % of Minimum Required → Stars Displayed

0% - 29%   → ☆☆☆ (No stars)
30% - 59%  → ★☆☆ (1 star earned)
60% - 99%  → ★★☆ (2 stars earned)
100%+      → ★★★ (Perfect score!)

Visual: ★ = Yellow/Gold (earned)
        ☆ = Gray (not earned)
```
